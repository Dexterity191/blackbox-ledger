# Blackbox Ledger

Public documentation for **Blackbox Ledger**, an internal accounting application operated by [Blackbox Research](https://blackboxresearch.dev/), a 501(c)(3) nonprofit organization.

This repository exists solely to host the application's public-facing legal documents and landing page, as required for OAuth integration with third-party financial services. The application itself is not open source and is not distributed.
